### VRAM 별 권장 모델

```
8GB VRAM  → 7B-8B 모델 (Q4 양자화)
16GB VRAM → 13B-14B 모델, 7B 고품질
24GB VRAM → 30B+ 모델, 13B 고품질
CPU Only  → 7B Q4 이하 (느리지만 가능)
```
