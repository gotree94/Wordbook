# 단어 암기장 (Wordbook) 앱

## 주요 기능

✨ **단어 암기장 앱이 완성되었습니다!**

### 기능 목록

1. **📁 파일 가져오기**
   - 텍스트 파일(.txt)로 단어장 추가
   - 형식: `영어단어 / 한국어뜻` (각 줄마다)
   - 예시:
     ```
     apple / 사과
     banana / 바나나
     cat / 고양이
     ```

2. **📚 단어장 관리**
   - 여러 단어장 생성 및 관리
   - 단어장별로 학습 진행
   - 단어장 삭제 가능

3. **✍️ 학습 모드**
   - 상단: 영어 단어 표시 (큰 파란색 카드)
   - 하단: 한국어 뜻 입력 영역
   - 스와이프로 이전/다음 단어 이동
   - 최대 100개 단어 학습

4. **⌨️ 텍스트 입력**
   - 키보드로 한국어 뜻 입력
   - TextField로 편리한 입력

5. **✅ 정답 확인**
   - "확인" 버튼으로 답 체크
   - 정답/오답 표시
   - 정답 시 초록색, 오답 시 빨간색 피드백

6. **📊 학습 결과**
   - 정답률 표시 (원형 차트)
   - 전체/정답/오답 개수 통계
   - 오답 노트 제공
   - 다시 학습하기 기능

## 필수 설정

### 특별한 설정 필요 없음! 🎉

바로 빌드해서 사용하실 수 있습니다.
(음성 인식 기능은 추후 추가 예정)

## 사용 방법

### 1. 텍스트 파일 준비
```text
apple / 사과
banana / 바나나
cat / 고양이
dog / 개
book / 책
```

### 2. 파일을 iPhone의 파일 앱에 저장

### 3. 앱에서 단어장 추가
- 메인 화면에서 `+` 버튼 터치
- 파일 선택
- 자동으로 단어장이 생성됨

### 4. 학습 시작
- 단어장 선택
- "학습 시작" 버튼
- 영어 단어를 보고 한국어 뜻 입력
- 키보드로 입력
- "확인" 버튼으로 정답 체크
- 스와이프로 단어 이동

### 5. 결과 확인
- 100개 완료 후 자동으로 결과 화면
- 정답률과 통계 확인
- 오답 노트로 틀린 단어 복습

## 주요 화면

1. **단어장 목록** - 저장된 모든 단어장 표시
2. **학습 카드** - 시원시원한 큰 글씨로 단어 학습
3. **결과 화면** - 아름다운 원형 차트로 성적 표시

## 기술 스택

- SwiftUI
- SwiftData (데이터 저장)
- Speech Framework (음성 인식)
- AVFoundation (오디오 처리)

## 팁

- 틀린 단어는 결과 화면의 오답 노트에서 다시 확인할 수 있습니다
- 키보드 위의 "완료" 버튼으로 키보드를 닫을 수 있습니다
